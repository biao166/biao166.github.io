# 安装软件

下载安装包，并安装

名称 官网 坚果云下载 备注
cmder cmder 下载 推荐full版，集成git
nvm nvm 下载 windows版本，不直接通过npm来安装
vscode vscode 下载 代码编辑器
notepad++ notepad++ 下载 文本编辑器

注意：部分软件是直接解压使用， 部分软件需要手动添加到 PATH

# 初始化

打开 cmder, 执行

# 检查 git
git -v

# 检查 nvm, 如提示未安装见下文
nvm -v

# 安装nodejs
nvm install latest

# 检查 node
node -v

# 检查 npm
npm -v

# npm 安装 nrm
npm install nrm -g --registry=https://registry.npm.taobao.org

# 展示可用源
nrm ls

# npm 使用淘宝源
nrm use taobao
